# SPA Mascotas

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```
La aplicación estara en `http://localhost:5173`.

## Construir

```bash
npm run build
```

## Despliegue a GitHub Pages

```bash
npm run deploy
```

## Estructura

- `src/components`
- `src/assets`
- `src/styles`
